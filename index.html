Gaussian Splat Studio

A 3D Gaussian Splatting viewer & editor written from scratch in Rust + WebGL2. Pick a scene to fly through it, erase splats, slice objects apart and move the pieces. View on GitHub ↗

How to use

🖥️ Desktop

DragLook around
WASDMove around
SpaceShiftMove up / down
Alt+ClickErase splats (or use the Erase tool)
P+DragSlice an object in two (or use the Slice tool)
ClickSelect a piece — drag the gizmo arrows to move it
OToggle octree view

📱 Touch

1 fingerDrag to look around
JoystickMove around the scene
PinchMove forward / backward
▲▼Move up / down
ErasePick the Erase tool, then tap or drag on the model
SlicePick the Slice tool, then drag a line across an object
TapSelect a piece — drag the gizmo arrows to move it
