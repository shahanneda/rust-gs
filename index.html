How to use

🖥️ Desktop

DragLook around
WASDMove around
SpaceShiftMove up / down
Alt+ClickErase splats (or use the Erase tool — hover previews in red)
P+DragSlice an object in two (or use the Slice tool)
ClickSelect an object — drag the gizmo arrows to move it
✨ MagicAI select: click any object, then extract, erase or recolor it
➕ AddDrop cake, fruit, Shahan or meshes into the scene
⌘ZUndo erase / move / recolor

📱 Touch

1 fingerDrag to look around
JoystickMove around the scene
PinchMove forward / backward
▲▼Move up / down
ErasePick the Erase tool, then tap or drag on the model
SlicePick the Slice tool, then drag a line across an object
TapSelect a piece — drag the gizmo arrows to move it
